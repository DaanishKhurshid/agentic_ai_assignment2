Orion Tech Support Hub

This project implements an Agentic AI RAG (Retrieval-Augmented Generation) system designed to provide automated technical support. The system uses LangGraph to manage a multi-step workflow (retrieval and generation) and provides an interactive interface via Streamlit.

Key Features
Agentic Workflow: Utilizes LangGraph for a structured, sequential process (Retrieval → Generation).

Vector Retrieval: Powered by ChromaDB for efficient, context-aware information retrieval from technical manuals.

Generative AI: Integrates with Google’s Gemini models to provide human-like, accurate technical support answers.

Interactive UI: A clean Streamlit dashboard for real-time user interaction.
